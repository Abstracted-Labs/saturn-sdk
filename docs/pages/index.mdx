The Saturn SDK is a set of tools that allow you to build apps with Multisigs on chain.

## Quick Start

<div className="steps-container">

### Install

To install the SDK it's pretty simple, you can just install the package with your favorite package manager:

<Tabs items={["pnpm", "npm", "yarn"]}>
  <Tab>```bash pnpm i @invarch/saturn-sdk ```</Tab>
  <Tab>```bash npm i @invarch/saturn-sdk ```</Tab>
  <Tab>```bash yarn add @invarch/saturn-sdk ```</Tab>
</Tabs>
